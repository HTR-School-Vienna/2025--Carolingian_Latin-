## Description

We produced ground truth data for a random selection of pages/foll. of Latin manuscripts written in Carolingian Minuscule (9th-12th ct.) in order to improve the Carolingian Minuscule Model 2025 (CMM 2025) for Transkribus. 
The course created 258 pages ground truth from the manuscripts listed below, following the transcription guidelines of CMM; here a reference to the official Zenodo Community: https://zenodo.org/communities/testo/records?q=&l=list&p=1&s=10&sort=newest
See a list of all transcribed pages / folios below.


### Origin of the data
This is a list of all 258 transcribed pages / folios.

Jena, Ms. Bos. q. 19, fol. 179v-183v, 185r-190r
Karlsruhe, BLB, Cod. Karlsruhe 504, fol. 187r-194r
Köln, Dombibliothek, Cod. 85, fol. 105r-109r, 114v-116v 
Leiden, Universitaire Bibliotheken, BPL 141, fol. 7v-16r
Paris, BnF, lat. 1979, fol. 1v-4v, 12v-18v
Paris, BnF, lat. 10911, fol. 3v-13r
St. Gallen, Stiftsbibliothek, Cod. 167, p. 11-19, 24-35
St. Gallen, Stiftsbibliothek, Cod. 561, p. 95-115
Vatikan, BAV, Pal. lat. 261, fol. 105r-115r
Vatikan, BAV, Reg. lat. 484, fol. 8r-15r
Wien, ÖNB, Cod. 89, fol. 2v-7v
Wien, ÖNB, Cod. 189, fol. 82v-89v
Wien, ÖNB, Cod. 427, fol. 75v-82v
Wien, ÖNB, Cod. 596, fol. 4r-12r, 10r-10v, 14v-16v
Wien, ÖNB, lat. 652, fol. 34r-44r

These folios were transcribed by Transkribus using both the private model CMM 2025 v2 (CER 4.77) and the public model Carolingian Minuscule Model CMM 9th-11th c. (CER 5.2), and then manually corrected by the group members.
